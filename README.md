# Daedalus Vis - TypeScript Algorithms Playground

A playground repository for exploring and implementing algorithms in TypeScript.

## About

This repository serves as a learning and experimentation space for algorithm implementations in TypeScript. It's designed to help developers practice, test, and optimize various algorithmic solutions.

## Structure

```
typescript/
├── algorithms/     # Algorithm implementations
├── types/     # Types explorations
└── tsconfig.json   # TypeScript configuration
```

## Getting Started

1. Clone the repository
2. Navigate to the `typescript` directory
3. Install dependencies (if any)
4. Run individual algorithm files with `ts-node` or compile with `tsc`

## Examples

Check out the `algorithms/` directory for various implementations:

- **repeated-chars.ts**: Determine if all characters in a string appear the same number of times

Check out the `types/` directory for TypeScript type explorations:

- **api-response.ts**: API response handling with generics, discriminated unions, and runtime type validation for external APIs
